# Basic Node.js Utilities ⚙️

A simple Node.js project that provides basic utility functions (like mathematical operations) and demonstrates how to structure and execute reusable scripts.

---

## 🎯 Project Purpose

This project was created to learn:

* Writing reusable utility functions in Node.js
* Organizing code into separate files
* Executing scripts using Node.js runtime

---

## 🛠️ Tech Stack

* Node.js
* JavaScript

---

## ⚙️ How to Run

```bash id="m9a2xz"
node script.js
```

---

## ✨ Features

* Basic mathematical utility functions
* Clean and simple code structure
* Easy to understand and extend

---

## 📂 Project Structure

```id="f7k1qp"
basic-nodejs-utils/
│
├── math.js       # Contains utility functions
├── script.js     # Executes and uses functions
└── .gitignore
```

---

## 🧠 Concepts Covered

* Function creation and reuse
* File separation in Node.js
* Importing modules using `require()`
* Code organization
